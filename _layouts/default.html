Projects

PlanExp
Web application to explore and visualize both traditional and single cell RNA-seq data for the planaria Schmidtea mediterranea. It provides several interactive visualization options made with the plotting library Plotly.js, a way to map gene expression onto a gene-protein network, a fully fledged Cytoscape.js network editor, and the ability to perform GO enrichment analyses.

Publication
Code

PlanNET
Prediction of protein-protein interactions for the planaria Schmidtea mediterranea using sequence homology information and machine learning. Creation of a web application to explore the predicted interactions in several ways, using the graph-based database manager Neo4j and the Django Python web framework.
Website
Publication
Code

PPaxe
Python and Web tool to extract protein-protein and genetic interactions from the scientific literature programatically. PPaxe uses a Random Forest Classifier and the Stanford CoreNLP to compute the necessary features. PPaxe is completely pattern-free, and is able to extract interactions independently of the identifiers used for the proteins and genes, thanks to the Name Entity Recognizer (NER) trained with several datasets.
Website
Publication
Code

RPGeNet v2
Complete rewrite of the Retinitis Pigmentosa Gene Interaction Network web application (RPGeNet) using Django and the database manager neo4j. RPGeNet was created to help users find new potential driver genes of retinitis pigmentosa and help better visualize the complex interaction network of the retina. The interaction database is formed by the merger of self-curated BioGRID, STRING and text-mining (PPaxe) interaction data.
Website
Publication
Code

NewIt
Django web application to read news from Spanish sources. The focus of the application is entirely on searches, and users subscribe to their preferred keywords instead of categories. Made for the Databases and Web Development Course of the MSc. for Health Sciences at UPF.
